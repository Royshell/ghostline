# 👻 GhostLine

> A lightweight drawing canvas for smooth, fading strokes — built for modern web apps.

![Demo](https://res.cloudinary.com/drehggl5c/image/upload/v1762705262/ghost-example-1_iwgyhe.gif)

GhostLine is a minimal and elegant canvas component designed for:

- ✍️ Freehand drawing
- 🧠 Visual thinking tools
- 📝 Annotations & overlays
- ✨ Ghost-like fading stroke effects

Built with performance and simplicity in mind.
